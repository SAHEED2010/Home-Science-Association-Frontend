# HSA Portal - School Management System

A modern, comprehensive School Management System built for the Home Science Association.

## Features

- **Public Website**: Home, About, Testimonials, Contact, and Admissions pages.
- **Role-Based Access**: Dedicated dashboards for Admins, Teachers, Students, and Parents.
- **Academic Management**: Classes, Courses, Attendance, Assignments, Exams, and Grades.
- **Finance**: Fee tracking, Invoicing, and Payment history.
- **Communication**: Announcements and Notifications.
- **Modern UI**: Dark mode support, responsive design, and animated components.

## Tech Stack

- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Lucide React](https://lucide.dev/)
- [Recharts](https://recharts.org/)

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run the development server:**
    ```bash
    npm run dev
    ```

3.  **Open the app:**
    Visit [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `src/app`: App Router pages and layouts.
- `src/components`: Reusable UI components.
- `src/lib`: Utility functions and mock data.
- `public`: Static assets (images, icons).
